# Santulan Therapy Clinic Website

A modern, lightweight, SEO-friendly, and AI-optimized static website for **Santulan Therapy Clinic**.

The project is built using **HTML5**, **CSS3**, and **Vanilla JavaScript** with no frontend framework or UI library, ensuring excellent performance and easy deployment.

---

# Features

- Modern healthcare-focused design
- Responsive layout
- Mobile-first approach
- Optimized for Google Lighthouse
- AI Search Engine Optimization
- Semantic HTML5
- Local Business Schema
- WhatsApp Appointment Booking
- Google Maps Integration
- Contact via Email
- Lazy Loading Images
- Progressive Web App Ready
- SEO Ready
- Accessible Design
- Lightweight Architecture
